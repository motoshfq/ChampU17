# Необхідні зображення

Для повного функціонування сайту, будь ласка, додайте наступні зображення в цей каталог:

1. hero-bg.jpg - фонове зображення для головного екрану
   - Рекомендований розмір: 1920x1080 пікселів або більше
   - Тематика: легкоатлетичні змагання, молоді спортсмени, біг, стрибки тощо
   - Темне зображення з хорошою контрастністю працюватиме найкраще
   - Бажано динамічне зображення з рухом

2. about-img.jpg - зображення для секції "Про змагання"
   - Рекомендований розмір: мінімум 800x600 пікселів
   - Тематика: спорт, змагання, спортивний манеж, молоді атлети
   - Високоякісне зображення з гарною композицією

3. pattern.png - паттерн для фону деяких секцій
   - Простий абстрактний паттерн
   - Мінімалістичний світлий дизайн (буде використовуватись з прозорістю)
   - Можна використати геометричні фігури, лінії або спортивні елементи
